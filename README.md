# Crypto Trader Performance Analysis (Fear & Greed Index)

This project analyzes how a crypto trader performs under different market sentiment conditions using:
 Hyperliquid trading data
 Bitcoin Fear & Greed Index

# Folder Structure

csv_files/     → Raw and processed datasets  
outputs/       → Charts and visualizations  
notebook_1.ipynb → Full data analysis and modeling  
ds_report.pdf  → Final project report  

# Objective
To study how trader profitability, win rate and risk change when market sentiment moves from Fear to Greed.

# Tools Used
 Python
 Pandas
 NumPy
 Matplotlib
 Seaborn
 Jupyter Notebook

# How to Run
Open `notebook_1.ipynb` in Jupyter or Google Colab and run all cells.

# Results
The trader performs better in Greed markets but shows higher drawdowns during Extreme Fear.
